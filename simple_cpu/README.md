# Simple CPU Design (Verilog RTL)

This project implements a basic 8-bit CPU architecture using Verilog.

## Modules
- ALU (Arithmetic Logic Unit)
- Register File
- Program Counter
- Instruction Execution

## Simulation
The design was simulated using:
- Verilog
- Icarus Verilog
- EDA Playground

Waveforms were verified using EPWave.
